# Personalization Service

An AI-driven microservice that personalizes the user experience with recommendations, user preferences, and behavior tracking.

## Features
- Personalized product/content recommendations
- User preference storage
- Behavior-driven learning
- API for cross-platform personalization

## Tech Stack
- Node.js + Express
- MongoDB
- TensorFlow.js / Scikit-learn
- OpenAPI Docs
